Class 1

社团大致介绍：成员，课题，最终的成果以及对升学的帮助。

Class 2

介绍怎样做一份科研报告：格式标准，符合招生官品味，美术设计。

Class 3

正式开题：讲解空气污染的原因及雾霾的产生过程，从目前已有的学科知识探讨全球气候的变化趋势。

Class 4

了解废气管理的法律法规及南京的化工厂排气情况。

Class 5

观看未来气候电影《后天》，及保护环境的纪录片《穹顶之下》。

Class 6

大气物理：凝胶颗粒在大气中的作用。

Class 7

流体力学：风的形成及如何利用传感器测试台风。

Class 8

计算机：如何利用远程终端控制数据的传输与处理。

Class 9

地理：了解华北平原的季风特征、四川盆地的气流特征及沿海地区的空气指标特征。（以山东日照为例看每日监测的气象数据）

Class 10

去实地参观化工厂及气象观测台，了解气象预报的完整流程。回来进行整理总结，准备下面的正式报告制作。

Class 11

制作科研报告：不同学科之间知识的串联，怎样体现在报告中。

Class 12

制作科研报告：完成报告，社长点评修改，美化完善。
